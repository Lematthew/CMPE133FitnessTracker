1. Write a ClassSpec.js for your Class.js to test, this ClassSpec.js should be in spec folder

2. In Spec.js
	a) describe(String description, function() {})
	b) in describe.function(), write your test cases: it(String description,function())
	c) in it.function(), write test body

3. In SpecRunner.html
	a) Include <script src="src/Class.js"></script>
	b) Include <script src="spec/ClassSpec.js"></script>

4. Run SpecRunner.html
